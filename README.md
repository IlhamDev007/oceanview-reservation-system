# 🌊 Ocean View Reservation System

A Java EE web application designed to manage room reservations for **Ocean View Resort**.  
The system improves efficiency by replacing manual booking records with a computerized reservation workflow, reducing booking conflicts and speeding up daily operations.

---

## 📌 System Overview

The **Ocean View Reservation System** supports secure staff access and provides a complete reservation workflow:  
**create reservations → view details → manage (search/edit/delete) → calculate bills and print invoices**.  
It also includes a dashboard summary to help staff quickly understand system status.

---

## ✅ Key Features

- Secure user login and session-based access  
- Add new room reservations  
- View reservation details  
- Manage reservations (search, edit/update, delete)  
- Billing: calculate total cost and print invoice  
- Dashboard summary (total reservations, available rooms, total revenue)  
- Help Section  
- Exit/logout  

---

## 🛠️ Technologies Used

- **Java EE** (Servlets, JSP)  
- **Apache Tomcat 9**  
- **MySQL + JDBC**  
- **HTML / CSS**  
- **Maven**  
- **JUnit**
